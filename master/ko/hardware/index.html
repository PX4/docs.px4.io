# 하드웨어 통합

PX4에 신규 자동조종장치와 주변장치를 통합하는 방법을 설명합니다.

  • 비행 콘트롤러 설계 참고
  • 제조업체 보드 지원 가이드
  • 비행 콘트롤러 포팅 가이드
  • 시리얼 포트 매핑
  • 기체
  • 장치 드라이버
  • 텔레메트리/모뎀 및 기타 통신 장치
  • 센서와 액츄에이터 I/O
  • RTK GPS (통합)
  • 짐벌 (마운트) 제어 설정
  • 보조 컴퓨터

TIP

다른 섹션에서는 지원 자동조종장치 종류, 주변 장치 사용법과 설정 방법을 설명합니다.
