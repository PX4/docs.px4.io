# Holybro 텔레메트리 라디오

Holybro는 PX4에서 사용 가능한 SiK 호환 텔레메트리 라디오를 제공합니다. 915MHz(미국) 및 433MHz(유럽) 버전을 사용할 수 있습니다.

# 구매:

  • Holybro 트랜시버 텔레메트리 라디오 V3433MHz (opens new window)
  • Holybro 트랜시버 텔레메트리 라디오 V3 915MHz (opens new window)
  • Holybro 텔레메트리 라디오 V3 (opens new window)
