# VTOL 설정

표준 설정을 수행합니다. 대기 속도 센서(선택 사항이지만 적극 권장됨)를 보정하고, RC 콘트롤러에 VTOL 전환 스위치를 할당하여야 합니다.

그런 다음 VTOL별 설정과 튜닝 작업을 합니다.

  • QuadPlane 설정
  • 후방 이동 튜닝
  • 속도 센서가 없는 VTOL
  • VTOL 날씨 풍향
