# Snapdragon 비행 자동조종장치 (단종됨)

WARNING

이 비행 컨트롤러는 단종되었습니다. 사용 방법에 대한 자세한 내용은 PX4 사용자 가이드 v1.11 (opens new window)을 참고하십시오.

PX4에서는 이 제품을 만들지 않습니다. 하드웨어 지원과 호환 문제는 제조사 (opens new window)에 문의하십시오.

Qualcomm Snapdragon Flight 플랫폼은 POSIX용 DSPAL API (opens new window)를 사용하여 QuRT 실시간 운영체제 DSP에서 PX4 Flight Stack을 실행하는 고급 자동조종장치/온보드 컴퓨터입니다. Pixhawk에 비교하여, 카메라와 WiFi, 고급 처리 능력, 기타 IO가 추가되었습니다.
