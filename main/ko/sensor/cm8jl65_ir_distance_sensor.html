# Lanbao PSK-CM8JL65-CC5 ToF 적외선 거리 측정 센서

Lanbao PSK-CM8JL65-CC5 (opens new window)는 0.17m-8m 범위와 밀리미터 분해능을 가진 초소형 IR 거리 센서입니다. UART/직렬 버스에 연결됩니다.

  • 크기: 38 mm x 18mm x 7mm
  • 중량: ≤10g

# 하드웨어 설정

PSK-CM8JL65-CC5 can be connected to any unused serial port, e.g.: TELEM2, TELEM3, GPS2 etc.

핀배열은 센서 하단에 레이블이 있습니다.

# 매개변수 설정

SENS_CM8JL65_CFG를 사용하여 LIDAR가 실행되는 직렬 포트를 설정합니다.

Note

If the configuration parameter is not available in QGroundControl then you may need to add the driver to the firmware:

distance_sensor/cm8jl65

In order to use the sensor for collision prevention you will further need to set the parameters SENS_CM8JL65_R_0 and CP_DIST. 자세한 내용은 충돌 방지를 참고하십시오.
