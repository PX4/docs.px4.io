# Airships

Experimental

WARNING

Support for airships is experimental. Maintainer volunteers, contribution of new features, new frame configurations, or other improvements would all be very welcome!

PX4 supports a single airship geometry with starboard, port and tail thrusters, and a thrust tilt server. The frame configuration is shown in Airframes Reference > Airship.
