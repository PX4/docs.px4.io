# PX4FLOW Smart Camera (Deprecated)

PX4 does not support the PX4Flow optical flow sensor from PX4 v1.13.

Documentation has been removed (if needed, see Legacy Docs for PX4Flow in v1.13 (opens new window)).

注解

PX4 v1.13 does not work with current PX4FLow firmware. PX4 may work with older PX4Flow firmware.
