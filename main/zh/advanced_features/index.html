# 高级功能

这部分包含了部分关于PX4自驾仪的高级功能：

  • Air Traffic Avoidance: ADS-B/FLARM/UTM
  • 机器视觉
    • 自主避障（需要机载计算机）
    • 安全降落（需要机载计算机）
    • 碰撞避免 （通过机载计算机或飞控的传感器）
    • 路径规划接口 （开发者接口）
    • 动态捕捉(MoCap)
    • 视觉惯性里程计(VIO)
  • 铱星/RockBlock卫通系统
  • 精准着陆
  • GPS载波相位差分技术(RTK GPS)
