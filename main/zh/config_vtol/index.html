# 垂直起降（VTOL）配置

First perform the Standard Configuration. As part of this you should calibrate the Airspeed sensor (optional, but highly recommended) and assign a VTOL transition switch to your RC controller.

Then perform VTOL-specific configuration and tuning:

  • QuadPlane Configuration
  • Back-transition Tuning
  • VTOL w/o Airspeed Sensor
  • VTOL Weather Vane
