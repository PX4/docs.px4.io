# Flight Modes

Flight modes provide autopilot support to make it easier to manually fly the vehicle, to automate common tasks such as takeoff and landing, to execute autonomous missions, or to defer flight control to an external system.

For information about flight modes available to specific frames see the following topics:

  • Flight Modes (Multicopter)
  • Flight Modes (Fixed-Wing)
  • Flight Modes (VTOL)
  • Flight/Drive Modes (Rover)

注解

The mode sub-topics in this section contain information that is common to all vehicles, but may not be relevant to the normal/default setup. Generally you should see the vehicle-specific topics first, and navigate to these generalized topics when it becomes relevant.
