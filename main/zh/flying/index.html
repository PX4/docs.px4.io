# 飞行

本节包含有关飞行（完全配置过的）自动驾驶仪控制机体的主题。 它涵盖了（自动驾驶仪辅助）手动飞行和完全自主飞行的任务规划。

首次飞行指南 - 确保您的第一次飞行愉快、有教育意义且安全的指南。

飞行入门课 - 如何使用遥控器手动控制机体。

任务 - 如何执行自主任务。

地理围栏 - 如何设置围栏。

集结点-如何规划安全返回降落点（集结点）。

飞行模式 - 摘要表 + 有关飞行模式的详细信息。

地形跟随/保持&范围辅助- 如何启用地形跟随。

提示

机体状态通知可以帮助您确定您的机体何时可以飞行（不能起飞，及其原因）。
