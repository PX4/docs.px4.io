# HEX/ProfiCNC Here+ RTK GPS (Discontinued)

Note

This GPS is no longer available for purchase but is still compatible with PX4.

Usage documentation can be found in PX4v1.11 docs (opens new window)

The Here+ RTK GPS receiver is a small, light and energy efficient RTK GPS module, based on the u-blox M8P. Using RTK, PX4 can get its position with centimetre-level accuracy, which is much more accurate than can be provided by a normal GPS.
