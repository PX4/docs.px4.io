# Plane Frame Builds

PX4 supports every imaginable plane geometry, including normal planes, flying wings, inverted V-tail planes etc. The complete set of supported configurations can be seen in Airframes Reference > Plane.

This section contains build logs/instructions for assembling and configuring a number of Plane frames.

  • Reptile Dragon 2 (ARK6X)
  • Turbo Timber Evolution (Pixhawk 4 Mini)
  • Wing Wing Z84 (Pixracer)
